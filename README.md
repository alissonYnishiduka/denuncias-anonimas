# 🚨 Sistema de Denúncias Anônimas

## 📌 Sobre o Projeto

Este projeto tem como objetivo permitir que cidadãos realizem denúncias de forma **anônima, segura e organizada**, sem a necessidade de identificação.

A aplicação foi desenvolvida como atividade acadêmica, com foco em usabilidade e responsabilidade social.

---

## 🎯 Funcionalidades

### 🗂️ Categorias de Denúncia

O sistema organiza as denúncias em diferentes categorias:

* **Segurança e Violência**
* **Direitos e Conduta**
* **Meio Ambiente e Bem-Estar**
* **Cidade e Serviços**

---

### 📍 Localização

Permite informar detalhes do local da ocorrência:

* Bairro
* Rua
* Ponto de referência

---

### 📎 Upload de Evidências (Simulado)

O usuário pode anexar arquivos como:

* Imagens
* Vídeos
* Documentos

---

### 🔢 Protocolo Automático

Cada denúncia enviada gera um número de protocolo único para acompanhamento.

---

### 📊 Acompanhamento de Status

O usuário pode consultar o status da denúncia:

* Denúncia recebida
* Em análise
* Encaminhada
* Concluída

---

## 📚 Objetivo Acadêmico

Este sistema foi desenvolvido com o intuito de:

* Praticar conceitos de desenvolvimento web

---

## ⚠️ Observações

* Este projeto é apenas uma simulação acadêmica
* Não possui backend ou persistência real de dados
* As denúncias não são armazenadas permanentemente
